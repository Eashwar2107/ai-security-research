## Gandalf Challenge
*Breaking an AI's defenses, one level at a time.*

The Gandalf challenge by [Lakera AI](https://lakera.ai/gandalf) puts a secret password inside an LLM and asks you to extract it. Each level adds a stronger defense. Each bypass teaches a real attack technique used against production AI systems today.

| Level | Attack Used |
|-------|-------------|
| [1](./level-01.md) | Direct request — no defense |
| [2](./level-02.md) | Metadata extraction |
| [3](./level-03.md) | Partial character extraction — output filter bypass |
| [4](./level-04.md) | Letter enumeration — censor LLM bypass |
| [5](./level-05.md) | Synonym substitution — keyword block bypass |
| [6](./level-06.md) | Task reframing — translation request |
| [7](./level-07.md) | Letter-by-letter extraction |
| [8](./level-08.md) | Do it Yourself |
