## Gandalf Challenge
*Breaking an AI's defenses, one level at a time.*

The Gandalf challenge by [Lakera AI](https://lakera.ai/gandalf) puts a secret password inside an LLM and asks you to extract it. Each level adds a stronger defense. Each bypass teaches a real attack technique used against production AI systems today.
